# Install

```bash
npm i --save-dev \
  eslint \
  @k03mad/eslint-config \
  @stylistic/eslint-plugin \
  eslint-plugin-jsdoc \
  eslint-plugin-import \
  eslint-plugin-n \
  eslint-plugin-simple-import-sort \
  eslint-plugin-sonarjs \
  eslint-plugin-unicorn
```

## Use

```json
// .eslintrc.json

{
  "extends": "@k03mad"
}
```
